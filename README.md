# AI Fusion Chat

An advanced AI-powered chat application featuring real-time messaging, AI assistant integration using Gemini, and voice chat capabilities.

## Features

- **Real-time Chat**: Chat with friends instantly using Socket.IO.
- **AI Assistant**: Integrate with Google Gemini for smart AI conversations.
- **Voice Chat**: Talk to the AI assistant with voice recognition.
- **Glassmorphic UI**: A modern, beautiful interface with dark mode support.
- **Authentication**: Secure signup/login including Google Auth.

## Tech Stack

- **Frontend**: React, Vite, TailwindCSS, Framer Motion
- **Backend**: Node.js, Express, MongoDB, Socket.IO
- **AI**: Google Gemini API
